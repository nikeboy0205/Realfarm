[레즐모 상시이벤트 안내]


[현재 진행중인 상시이벤트]

- 뽑기이벤트 (기한 : 무제한)


[참여 조건]

- 레즐모 톡방 맴버 전원 참여가능

- 개인당 이틀(24시간)마다 1번씩 참여가능( tip ! 관리자가 심심할때 리셋해요! )

- 페이지 맨 아래 주의사항 필독

[뽑기이벤트 당첨금]

- 당첨되는 모든분들께 1억골드 지급 (이틀마다 계속 중복참여 가능)



[뽑기이벤트 참여방법↓]





1. 레즐모 톡방에 !뽑기 를 입력합니다.








2. 당첨이 안됬을경우 봇이 위 사진처럼 말합니다.









3. 만약 당첨이 되면 봇이 위 사진처럼 말합니다. 당첨되었다면 위 화면을 캡쳐합니다.








4. 당첨된 메세지를 캡쳐후 톡방 오른쪽 위에있는 짝대기 3개모양 버튼(설정)을 누릅니다.








5. 아래에 태어난김에사는남자(방장) 닉네임을 클릭합니다.







6. 1:1오픈채팅 버튼을 눌러 방장에게 캡쳐한 당첨사진을 보냅니다.



캡쳐본 보낸후 여유롭게 기다려주세요~ 확인하는대로 바로 당첨금 지급해드립니다!^^



※주의사항※

- 상시이벤트에 봇 기능을 악용해서 참가하시면 이벤트 참여자격 영구박탈합니다.
영구박탈당하면 절대 안풀어드려요. 톡방 맴버분들 위해서 사비 들여가며 만든 이벤트인데 악용하지 말아주세요.
